## Overview

The Tab class is the controlling mechanism for a browser target. For most users, a "target" is a tab, but it can also be an iframe, service worker, or background script.

When you open a new window using `browser.get(..., new_window=True)`, your URL opens in a new window, which is represented as a Tab. When you browse to another page, the Tab object remains the same.

## Navigation methods

    self,
    url: str = "about:blank",
    new_tab: bool = False,

<ParamField path="url" type="str" default="about:blank">

<ParamField path="new_window" type="bool" default="False">

</ParamField>

<ResponseField name="tab" type="Tab">

## Finding elements

### find()

Find a single element by text content.

```python
async def find(
    self,
    text: str,
    best_match: bool = True,
    return_enclosing_element: bool = True,
    timeout: Union[int, float] = 10,
) -> Element
```

<ParamField path="text" type="str">
  Text to search for (including script contents).
</ParamField>

<ParamField path="best_match" type="bool" default="True">
  When True, returns the element with the most similar text length. When False, returns the first match.
</ParamField>

<ParamField path="return_enclosing_element" type="bool" default="True">
  Return the containing element instead of the text node.
</ParamField>

<ParamField path="timeout" type="float" default="10">
  Timeout in seconds.
</ParamField>

<ResponseField name="element" type="Element">
  The found element.
</ResponseField>

## Examples

### Basic navigation and element interaction

```python
import zendriver as zd

browser = await zd.start()
tab = await browser.get("https://example.com")

# Find and click a button
button = await tab.find("Click me")
await button.click()

# Select element by CSS
link = await tab.select("a.important-link")
await link.click()
```

### Execute JavaScript

```python
tab = await browser.get("https://example.com")

# Simple evaluation
title = await tab.evaluate("document.title")
print(f"Page title: {title}")

# Dump window object
window_props = await tab.js_dumps("window")
print(window_props)
```

### Screenshots

```python
tab = await browser.get("https://example.com")

# Save viewport screenshot
await tab.save_screenshot("page.png", format="png")

# Save full page screenshot
await tab.save_screenshot("fullpage.jpg", full_page=True)

# Get base64 screenshot
data = await tab.screenshot_b64()
```

### Wait for elements

```python
tab = await browser.get("https://example.com")

# Wait for selector
element = await tab.wait_for(selector=".dynamic-content")

# Wait for text
element = await tab.wait_for(text="Loading complete")
```

### Window management

```python
tab = await browser.get("https://example.com")

# Maximize window
await tab.maximize()

# Set custom size
await tab.set_window_size(left=100, top=100, width=1920, height=1080)

# Scroll page
await tab.scroll_down(amount=50, speed=1000)
```

## Notes

- Tab objects persist across page navigations
- Always use `await` when calling Tab methods
- Use `await tab` to update references and let the script "breathe"
- For text searches, use `find()` with `best_match=True` for better results
- Screenshots work best after ensuring the page is fully loaded
